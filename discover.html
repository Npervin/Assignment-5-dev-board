Question-1: What are the different ways to select an element in the DOM?

There are many ways to select elements in the DOM like - selecting by ID, selecting by classname, by tag name, by using query selector, body element, active element, by name attribute.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML → is used to insert or read HTML tags inside an element. innerText → is used for visible text and ignore hidden elements. textContent → is used for all text even hidden and no HTML interpretation.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a parent element listens for events on its child elements, even if they are dynamically added later. Instead of adding individual event listeners to each child, we add one listener to the parent.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event starts from the target element and propagates (bubbles) up through its parent elements to the root (document).

Question-5: How do you create, add, and remove elements using JavaScript?

To create elements we should use (document.createElement), to add appendChild() (Add at the End) and prepend() (Add at the Beginning), to remove we should use element.remove() using JavaScript.
